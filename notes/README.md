## 1. Lint the code

> ### Ruff Checks

$ ruff check --fix ./path/to/file.py  
_Why is it important?_

* **Rating the code**
* **Shows you linting errors and what to fix**  

> ### Pylint Checks

$ pylint ./path/to/file.py  
_Why is it important?_

* **Rating the code**  
* **Shows you linting errors and what to fix**
